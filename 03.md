# File based routing

## Nested routes

- Map segments of the URL to components in the UI
  app
  ├── root.jsx
  └── routes
  | ├── sales
  | │ ├── overview.jsx
  | │ ├── subscriptions.jsx
  | │ ├── invoices.jsx
  | │ ├── invoices
  | │ │ ├── $invoiceId.jsx
  | │ │ └── index.jsx
  | │ ├── customers.jsx
  | │ ├── deposits.jsx
  | │ ├── index.jsx
  | └── sales.jsx

**app/routes/sales.jsx**

```tsx
import { Outlet } from "@remix-run/react";

export default function Sales() {
  return (
    <div>
      <h1>Sales</h1>
      <SalesNav />
      <Outlet />
    </div>
  );
}
```
